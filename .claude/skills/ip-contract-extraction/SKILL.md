# IP合同提取 Skill

## 概述
本skill提供IP授权合同（特别是漫画作品授权合同）的关键信息提取能力，帮助从合同文本中准确提取193个字段的结构化数据。

## 适用场景
- 漫画作品授权许可合同分析
- IP版权采购合同梳理
- 合同条款结构化提取
- 合同关键信息Excel化

## 核心能力

### 1. 合同结构识别
- 识别合同审批表、正文、附录三大部分
- 识别合同章节（一、二、三... 十七）
- 识别子章节（（一）、（二）...）
- 识别条款编号（1、2、3...）

### 2. 字段提取规则
按6大类提取193个字段：
- **合同基本信息**（27个字段）：编号、标题、主体、生效期等
- **合同类型信息**（3个字段）：主合同/补充合同/终止合同
- **权利项信息**（25个字段）：作品类型、采购区域、期限、独家性等
- **结算信息-基本**（4个字段）：预估金额、币种、价格形式
- **结算信息-详情**（110个字段）：结算类型、买断金额、分成比例等（重复2组）
- **合同管理字段**（19个字段）：作者、优先权、转签等
- **项目管理字段**（6个字段）：法务编号、是否录入CAMS等

### 3. 结算类型智能判断
根据关键词和逻辑判断6种结算类型：
1. 买断
2. 买断+奖励分成
3. 保底分成
4. 分成
5. 买断阶梯分成/阶梯分成
6. 其他

### 4. 币种识别和编码
支持12种币种的识别和转换：
- 人民币（CNY） → 12
- 美元（USD） → 1
- 港币（HKD） → 2
- 日元（JPY） → 3
- 等

### 5. 日期格式标准化
统一转换为 YYYY-MM-DD 格式

## 使用方法

在contract-extractor Agent中调用本skill：

```markdown
我需要提取IP授权合同的关键信息。请使用 @ip-contract-extraction skill来：

1. 识别合同结构
2. 提取193个字段
3. 判断结算类型
4. 转换币种编码
5. 生成JSON数据
```

## 关键注意事项

1. **必填字段优先**：优先提取合同编号、主体、金额等核心字段
2. **结算类型准确**：仔细阅读"收入结算支付"章节，准确判断结算模式
3. **币种转换正确**：CNY→12, USD→1, 严格按编码表转换
4. **日期格式统一**：所有日期转换为YYYY-MM-DD
5. **特殊条款识别**：注意自动续期、优先权、转授权等特殊约定
