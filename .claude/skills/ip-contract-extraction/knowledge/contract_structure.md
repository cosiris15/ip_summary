# 合同结构说明

## 总体结构

漫画作品授权许可合同通常包含三大部分：

```
合同文档
├── 第一部分：合同审批表（元数据表格）
├── 第二部分：合同正文（17个章节）
└── 第三部分：附录（可选）
```

## 第一部分：合同审批表

### 位置特征
- 位于合同开头
- 通常是HTML table格式
- 标题为"合同审批表"

### 关键字段

| 字段名 | 提取目标 | 示例 |
|--------|---------|------|
| 合同编号 | 合同唯一标识 | T-277-AUT-20150331-01 |
| 涉及金额 | 合同总金额 | CNY 0.00 |
| 付款方式 | 结算周期描述 | 按采购量结算：每月按页/月来结算 |
| 合同有效期 | 起止日期 | （约定）2015-04-01到2025-03-31 |
| 涉及单位、负责人 | 对方主体信息 | 周智延 460004198707230019 |
| 合同状态 | 审批状态 | 审批结束-待盖章 |
| 经办部门 | 内部部门 | 动漫业务部 |
| 经办人 | 内部联系人 | xshen（沈翔） |

## 第二部分：合同正文

### 标题和主体信息

```markdown
# 漫画作品授权许可合同

甲方（被授权方）：深圳市腾讯计算机系统有限公司
乙方（授权方）：周智延（笔名：鼠叔）

生效日期：【2015】年【4】月【1】日
```

**提取字段**：
- 合同标题
- 我方主体（甲方）
- 对方主体名称（乙方）
- 生效期

### 章节结构

#### 一、授权许可的漫画作品
**关键信息**：
- 作品名称：《老师是无赖》
- 作品形式：故事漫画、四格漫画等

**提取字段**：
- 采购作品名称
- 作品类型（2=静态漫画，3=动态漫画）

#### 二、漫画作品的基本要求
**关键信息**：
- 交付数量：每月不少于40幅
- 规格要求：1000k以内，600dpi以上

**提取字段**：
- 画稿类型（根据规格判断）

#### 三、著作权的归属和行使
**关键信息**：
- 著作权归属：乙方享有
- 授权性质：甲方专有、独家运营

**提取字段**：
- 作品类型（根据著作权描述判断）

#### 四、发表和修改特别约定
**关键信息**：
- 发表渠道：仅限腾讯动漫平台
- 修改权限：甲方可要求修改

#### 五、授权范围和期限 ⭐
**关键信息**：
- 授权区域：全球范围 / 中国大陆
- 授权性质：独占授权 / 非独占授权
- 授权期限：10年，自2015-04-01至2025-03-31
- 授权语言：中文及其他各种语言 / 简体中文
- 授权范围：全权授权

**提取字段**：
- 采购区域
- 采购语言
- 采购期限-开始时间
- 采购期限-到期时间
- 是否独家（1=独家，0=非独家）
- 是否转授权（根据"转授权权利"判断）
- 是否自动续期（根据"授权期满后X年"判断）
- 续期时间、续期次数

#### 六、甲方权利义务
**关键信息**：
- 运营权利：专有、独家运营
- 标识使用：可标注"腾讯独家"
- 审核义务：按时审核作品
- 支付义务：按约定支付费用

#### 七、乙方权利义务
**关键信息**：
- 著作权：享有著作权
- 创作义务：保质保量完成创作
- 合规要求：不得侵权、不得包含违禁内容

#### 八、著作权的保护
**关键信息**：
- 维权责任：甲乙双方共同负责
- 维权费用：2万以内甲方承担，超过部分1:1分担

#### 九、著作权的运营
**关键信息**：
- 运营方式：形象授权、在线阅读、出版、改编等
- 转授权：甲方可授权第三方

#### 十、收入结算支付 ⭐⭐⭐
**最关键章节，包含3个子章节**

##### （一）费用基准
包含多种结算类型：

**1. 连载费用**
- 稿费：250元/幅
- 连载收入分成：实际收入的10%
- 纯分成模式：实际收入的60%（当单月实际收入的50%高于当月稿费时）

**识别特征**：
- 关键词："稿费"、"元/幅"、"分成"
- 结算类型：买断+奖励分成（或纯分成）

**2. 形象授权费用**
- 金额：2000-10000元（一次性）

**3. 动画运营分成**
- 自行运营：纯利润的7%
- 授权第三方：授权收入的15%

**4. 游戏运营分成**
- 授权第三方：授权收入的15%

**5. 作品出版分成**
- 线下出版：超过1万本部分版税的50%
- 线下杂志：50%分成
- 周边衍生读物：销售收益另行协议

**6. 作品转授权分成**
- 分成比例：甲方80%，乙方20%

**7. 其他运营分成**
- 电影、电视剧等：另行签署补充协议

##### （二）收入结算
- 固定付费：按月结算
- 分成收入：每半年结算一次

**提取字段**：
- 结算周期（每月结算/每季度结算/每半年结算）

##### （三）收入支付
- 收款账户信息
- 税务约定：税前标准，代扣代缴
- 币种：人民币

**提取字段**：
- 币种（12=人民币）

#### 十一、保密约定
**关键信息**：
- 保密内容：人物设定、故事大纲、运营情况、费用标准等
- 保密期限：长期保密

#### 十二、违约责任
**关键信息**：
- 转委托违约：违约金30%
- 产量违约：连续3个月少于32幅
- 侵权违约：审核不通过，终止连载
- 未经同意发表：暂停连载或解约
- 已授权第三方：解约+违约金

#### 十三、不可抗力
#### 十四、沟通联络
#### 十五、适用法律
#### 十六、争议解决
#### 十七、协议效力

## 第三部分：附录

包含各种附件：
- 连载费用结算单样例
- 连载合作分成结算单样例
- 著作权运营收入分成结算单样例

## 提取流程建议

1. **第一步：扫描审批表** → 提取基本信息（编号、金额、有效期）
2. **第二步：识别主体** → 提取甲方、乙方信息
3. **第三步：读取"一、授权许可的漫画作品"** → 提取作品名称、形式
4. **第四步：读取"五、授权范围和期限"** → 提取区域、语言、期限、独家性
5. **第五步：重点分析"十、收入结算支付"** → 判断结算类型、提取金额和分成比例
6. **第六步：扫描其他章节** → 补充特殊条款信息

## 常见变体

### 补充合同
- 标题可能包含"补充协议"字样
- 会引用原合同编号
- 可能只修改部分条款

### 终止合同
- 标题包含"终止协议"字样
- 明确终止日期和原因

### 框架合同
- 标题包含"框架协议"字样
- 约定合作框架，具体项目另行签订

## 特殊条款识别

### 自动续期
**识别模式**：
- "授权期满后【X】年"
- "自动续期【X】年"
- "续期【X】次"

**示例**：
> 授权期满后三年，甲方仍有权按照本协议的约定继续享有漫画作品的专有授权权利。

**提取结果**：
- 是否自动续期：1（是）
- 续期时间：3年

### 优先权条款
**识别模式**：
- "优先投资权"
- "优先续约权"
- "优先受让权"

### 转授权条款
**识别模式**：
- "转授权权利"
- "有权对第三方进行转授权"
- "甲方可授权第三方"
